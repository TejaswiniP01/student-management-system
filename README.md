# Student Course Management System

## 📌 Description
This project is a desktop application built using Python Tkinter and MySQL.

## ⚙️ Features
- Add Students
- Add Courses
- Register Students
- Seat Availability Check
- View Registered Data

## 🛠️ Technologies Used
- Python (Tkinter)
- MySQL

## ▶️ How to Run
1. Install MySQL
2. Create database:
   CREATE DATABASE student_db;

3. Create tables:
   Student, Course, Registration

4. Run:
   python student_system.py
